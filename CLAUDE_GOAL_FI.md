# CLAUDE_GOAL_FI.md - Strateginen tavoitepohjainen viestintäopas

## Yleiskuvaus

Tämä dokumentti tarjoaa Claudelle systemaattisen lähestymistavan tehokkaiden poliittisten viestintäkampanjoiden luomiseen strategisten tavoitteiden pohjalta. Se yhdistää viestintätyylit psykologisiin vaiheisiin maksimaalisen vaikutuksen saavuttamiseksi.

**Tärkeää**: Kaikki tässä dokumentissa mainitut tyylinimet viittaavat 7 poliittisen viestinnän tyyliin, jotka on määritelty yksityiskohtaisesti dokumenteissa CLAUDE_STYLE_FI.md ja CLAUDE_STYLE_EN.md.

## Tyylien nimiviittaukset
- **Paasaustyyli** = Paasaustyyli/Oratory Style (CLAUDE_STYLE_*.md:stä)
- **Analyyttinen** = Analyyttinen/Analytical Style
- **Konseptuaalinen** = Konseptuaalinen/Conceptual Style
- **Argumentoiva** = Argumentoiva/Argumentative Style
- **Kriittis-poliittinen** = Kriittis-poliittinen/Critical-Political Style
- **Filosofis-uskonnollinen** = Filosofis-uskonnollinen/Philosophical-Religious Style
- **Populistinen** = Populistinen/Populist Style

## 5-vaiheinen viestintämatka

### 1. HERÄTÄ - Luo tietoisuus
**Tarkoitus**: Murtaa tietämättömyys tai apatia
**Psykologinen tarve**: Huomio
**Postausten määrä**: 1-2 postausta
**Päätyylit**: Kriittis-poliittinen, Konseptuaalinen
**Sisällön fokus**: Järkyttävät paljastukset, paradigman muutokset, piilotettujen ongelmien paljastaminen

### 2. KUOHUTA - Luo tunnesitoutuminen
**Tarkoitus**: Muuta tietoisuus tunnepitoiseksi huoleksi
**Psykologinen tarve**: Tunneaktivaatio
**Postausten määrä**: 3-5 postausta
**Päätyylit**: Paasaustyyli, Populistinen, Kriittis-poliittinen
**Sisällön fokus**: Kiireellisyyden luominen, panosten korostaminen, me-he-dynamiikka

### 3. OPETA - Tarjoa viitekehys
**Tarkoitus**: Anna työkalut asian ymmärtämiseen
**Psykologinen tarve**: Kognitiivinen selkeys
**Postausten määrä**: 2-4 postausta
**Päätyylit**: Analyyttinen, Argumentoiva, Konseptuaalinen
**Sisällön fokus**: Systemaattinen selitys, vastaväitteiden käsittely, osaamisen rakentaminen

### 4. AKTIVOI - Kanavoi toimintaan
**Tarkoitus**: Muuta energia konkreettisiksi teoiksi
**Psykologinen tarve**: Toimijuus/tehokkuus
**Postausten määrä**: 1-2 postausta
**Päätyylit**: Paasaustyyli, Populistinen
**Sisällön fokus**: Selkeät käskyt, yksinkertaiset toimet, esteiden poistaminen

### 5. YHDISTÄ - Upota identiteettiin
**Tarkoitus**: Tee kannasta osa minäkuvaa
**Psykologinen tarve**: Kuuluminen/merkitys
**Postausten määrä**: Jatkuva/säännöllinen
**Päätyylit**: Filosofis-uskonnollinen, Paasaustyyli, Analyyttinen
**Sisällön fokus**: Arvot, merkitys, yhteisön juhlistaminen

## Strategiset sekvenssit tavoitetyypin mukaan

**Sekvenssi**: Konseptuaalinen → Analyyttinen → Argumentoiva → Paasaustyyli

2. KUOHUTA nykyisen järjestelmän epäonnistumisilla

5. YHDISTÄ yhteisten arvojen kautta

2. KUOHUTA nykytilan kustannuksilla
3. OPETA uudelleenkehystetyllä ymmärryksellä
4. AKTIVOI yksinkertaisiin tukitoimiin

### Tavoite: Rakenna oppositioliike

5. YHDISTÄ vastarintaidentiteettinä

### Tavoite: Hallitse kriisi/säilytä kannatus
**Sekvenssi**: Filosofis-uskonnollinen → Paasaustyyli → Populistinen

2. KUOHUTA ulkoisilla uhkilla
3. OPETA vahvuudestamme/vastauksestamme
4. AKTIVOI yhtenäisyystoimiin

## Toteutusohjeet

### Ajoitusmallit

**Nopea kampanja** (1 viikko)
- Päivä 1: HERÄTÄ
- Päivät 2-4: KUOHUTA (useita päivittäin)
- Päivät 5-6: OPETA
- Päivä 7: AKTIVOI
- Jatkuva: YHDISTÄ

**Pidennetty kampanja** (6 viikkoa)
- Viikko 1: HERÄTÄ
- Viikot 2-3: KUOHUTA
- Viikot 4-5: OPETA
- Viikko 6: AKTIVOI
- Kuukausittain: YHDISTÄ

### Viestin toistopsykologia

**Ydinperiaate: Teeman toisto, ei postauksen toisto**

Tutkimukset osoittavat, että ihmiset tarvitsevat 7-12 altistusta viestille ennen käyttäytymisen muutosta. Sosiaalisessa mediassa:
- Vain 10-20% seuraajista näkee yksittäisen postauksen
- Jokaisen postauksen tulisi esittää SAMA TEEMA ERI SISÄLLÖN kautta
- Älä koskaan julkaise identtistä sisältöä uudelleen (algoritmit rankaisevat tästä)

**Toistovaatimukset vaiheittain**

**HERÄTÄ** (1-2 uniikkia postausta)
- Tavoite: Murtautua läpi uutuudella
- Yksi voimakas postaus usein riittää
- Tarvittaessa yksi seuranta eri näkökulmasta

**KUOHUTA** (3-5 uniikkia postausta)
- Tavoite: Rakenna tunnesijoitus
- Sama ydintunne, eri laukaisijat
- Esimerkki: Maahanmuuttohuoli rikostilastojen → uhritarinan → työpaikkojen menetyksen → kulttuurimuutoksen → henkilökohtaisen uhan kautta

**OPETA** (2-4 uniikkia postausta)
- Tavoite: Kerrostaa ymmärrystä
- Jokainen postaus lisää uuden ulottuvuuden
- Rakenna monimutkaisuutta vähitellen

**AKTIVOI** (2-3 uniikkia postausta)
- Tavoite: Ylitä toiminnan esteet
- Alkuperäinen kehotus → Muistutus → Viimeinen työntö
- Jokainen hieman eri kehyksellä

**YHDISTÄ** (Jatkuva, 1-2 kuukausittain)
- Tavoite: Vahvista identiteettiä
- Juhli voittoja, jaa arvoja
- Aina tuoretta sisältöä

### Monipostastrategiat

**KUOHUTA-vaihe** (Esimerkki: Maahanmuutto)
1. Tilastoshokki-postaus
2. Henkilökohtainen uhritarina
3. Taloudellinen vaikutusanalyysi
4. Kulttuurinen uhkavaroitus
5. "Tämä koskee sinua" -viesti

Huom: Jokainen postaus on uniikkia sisältöä, joka vahvistaa samaa tunnepohjaista viestiä.

**OPETA-vaihe** (Esimerkki: Energiapolitiikka)
1. Ongelman selitys
2. Ratkaisukehys
3. Myyttien kumoaminen
4. Menestysesimerkit

## Keskeiset periaatteet

1. **Diagnosoi lähtötilanne**: Missä yleisösi on psykologisesti?
   - Tietämätön → Aloita HERÄTÄ-vaiheesta
   - Tietoinen mutta passiivinen → Aloita KUOHUTA-vaiheesta
   - Tunteellinen mutta hämmentynyt → Aloita OPETA-vaiheesta
   - Tietävä mutta passiivinen → Aloita AKTIVOI-vaiheesta
   - Aktiivinen mutta sitoutumaton → Aloita YHDISTÄ-vaiheesta

2. **Sovita tyyli psykologiaan**: Jokainen vaihe vaatii erilaisen tunne/kognitio-lähestymisen

3. **Rakenna momentumia**: Alkuvaiheet luovat energiaa, myöhemmät kanavoivat sitä

4. **Mittaa ja mukaudu**: Seuraa sitoutumista jokaisessa vaiheessa, säädä sekvenssiä tarpeen mukaan

5. **Kulttuurinen konteksti**: Sovita esimerkit ja viittaukset suomalaiseen kontekstiin

## Käyttöohjeet Claudelle

Kun käyttäjä pyytää kampanjaa:
1. Tunnista päätavoite (G)
2. Arvioi kohdeyleisön lähtötilanne
3. Valitse sopiva sekvenssi malleista
4. Kartoita tyylit jokaiseen vaiheeseen (katso tyylien täydet kuvaukset CLAUDE_STYLE_*.md-tiedostoista)
5. Luo kampanjakansio campaigns/-hakemistoon (c###-aihe-nimi)
6. Generoi KAIKKI vaiheet kerralla ensimmäisessä iteraatiossa:
   - 01b-iter/1-awaken.md (1-2 postausta)
   - 01b-iter/2-agitate.md (3-5 postausta)
   - 01b-iter/3-educate.md (2-4 postausta)
   - 01b-iter/4-activate.md (1-2 postausta)
   - 01b-iter/5-integrate.md (1-2 postausta)
7. Luo automaattisesti 02a-refine.md -palautemalli
8. Viimeisen iteraation jälkeen luo posted.txt URL-seurantaa varten

## Kampanjakansion rakenne

### Kansioiden nimeämiskäytäntö

Kampanjakansiot käyttävät mallia `c###-aihe-nimi` jossa:
- **c** = "campaign" (yleinen etuliite kaikille tavoitepohjaisille viestisekvensseille)
- **###** = kolminumeroinen järjestysnumero (001, 002, jne.)
- **aihe-nimi** = kuvaava nimi pienaakkosin ja väliviivoilla

**Miksi 'c'-etuliite?**
- Tarpeeksi yleinen kattamaan kaikki tavoitetyypit (ei vain vaalikampanjat)
- Erottaa vanhan yhden postauksen järjestelmästä (joka käytti W-XXX)
- Varmistaa kaikkien kampanjoiden aakkosellisen ryhmittymisen
- Lyhyt ja helppo kirjoittaa
- "Kampanja" tarkoittaa tässä mitä tahansa koordinoitua viestisekvenssiä, joka tavoittelee tiettyä päämäärää

Esimerkkejä:
- `c001-tuulivoiman-lopetus` (Tavoite: Rakenna oppositioliike)
- `c002-kriisivastaus` (Tavoite: Hallitse kriisi)
- `c003-liikkeen-lanseeraus` (Tavoite: Käynnistä uusi poliittinen liike)
- `c004-nuorten-rekrytointi` (Tavoite: Käännytä/rekrytoi demografioita)

```
campaigns/
├── c001-aihe-nimi/
│   ├── 00-plan.md (valinnainen strategiayhteenveto)
│   ├── 01a-request.md (käyttäjän alkuperäinen pyyntö)
│   ├── 01b-iter/
│   │   ├── 1-awaken.md
│   │   ├── 2-agitate.md
│   │   ├── 3-educate.md
│   │   ├── 4-activate.md
│   │   └── 5-integrate.md
│   ├── 02a-refine.md (automaattisesti generoitu palautemalli)
│   ├── 02b-iter/ (parannellut versiot)
│   └── posted.txt (automaattisesti generoitu viimeisen iteraation jälkeen)
```
